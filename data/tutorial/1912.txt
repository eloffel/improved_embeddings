skflow has been moved to tensorflow.

   skflow has been moved to [94]http://github.com/tensorflow/tensorflow
   into contrib folder specifically located [95]here. the development will
   continue there. please submit any issues and pull requests to
   tensorflow repository instead.

   this repository will ramp down, including after next tensorflow release
   we will wind down code here. please see instructions on most recent
   installation [96]here.
