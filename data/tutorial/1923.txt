paralex

paraphrase-driven learning for open id53

about

   paralex is a system that learns to answer questions. paralex uses a
   large question-paraphrase corpus as a source of supervision, and then
   answers questions using [3]a massive database of facts extracted from
   the web.

   to learn more about paralex, read [4]paraphrase-driven learning for
   open id53 (fader et al., acl 2013).

data

   paralex learns from a collection of 18 million question-paraphrase
   pairs scraped from wikianswers.
     * please read the [5]readme.txt file before downloading.
     * download the data here: [6]wikianswers-paraphrases-1.0.tar.gz (543m
       compressed, about 4g uncompressed)

code

   you can download the code and data used to train and evaluate paralex.
   the download contains all data (including the wikianswers paraphrase
   corpus and the [7]reverb database) and is quite large (11gb
   uncompressed). please read the readme file before downloading to make
   sure that it is what you need.
     * [8]paralex evaluation readme
     * [9]paralex-evaluation.tar.bz2 (3.4gb compressed, 11gb uncompressed)
