readme.md

easyid35

   easyid35 is a id35 parser created by mike lewis.

   if you use easyid35 in your research, please cite the following paper:
   a* id35 parsing with a supertag-factored model, mike lewis and mark
   steedman, emnlp 2014

   pre-trained models are available from:
   [73]https://drive.google.com/#folders/0b7ay6pgz8lc-ngvocufxnu5vwxc to
   train new models, follow the instructions in training/readme

   basic usage: java -jar easyid35.jar --model model

   for n-best parsing: java -jar easyid35.jar --model model --nbest 10

   to parse questions, use: java -jar easyid35.jar --model model_questions
   -s -r s[q] s[qem] s[wq]

   if you want pos/ner tags in the output, you'll need to supply them in
   the input, using the format word|pos|ner. to get this format from the
   c&c tools, use the following: echo "parse me" | candc/bin/pos --model
   candc_models/pos | candc/bin/ner -model candc_models/ner -ofmt
   "%w|%p|%n \n" | java -jar easyid35.jar -model model_questions -i
   posandnertagged -o extended

   to get boxer-compatible prolog output, use: echo "parse me" |
   candc/bin/pos --model candc_models/pos | candc/bin/ner -model
   candc_models/ner -ofmt "%w|%p|%n \n" | java -jar easyid35.jar -model
   model -i posandnertagged -o prolog -r s[dcl]
