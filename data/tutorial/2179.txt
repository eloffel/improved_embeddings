a convolutional attention network for extreme summarization of source code

   a convolutional attentional neural network for the summarization of
   source code tokens in a short and consise method name.

dataset

   the dataset consists of the parsed source code of the top 11 projects.
   for information about the structure, please see the [1]dataset.txt.
   [2]download

source code

   the source code for this work, can be found [3]here.
