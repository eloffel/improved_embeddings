a tutorial on using the vizier memory-based learning system

   by: jeff schneider, mary soon lee, andrew moore

   description: a tutorial on using the windows vizier software, which
   provides a gui for doing fast and autonomous locally weighted learning
   and nearest neighbor style learning. the vizier software itself is
   available for free download at [74]http://www.cs.cmu.edu/~awm/vizier,
   where you will find more details.
