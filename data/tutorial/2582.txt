     * [6]java c&c

   download

   the gzipped tar file for download is 119m. the unpacked version
   requires 574m (excluding the .tgz file itself).
     * [7]java-candc-v0.95.tgz [14 aug 2015]

   install

   create a new directory called java-candc, move the gzipped tar file to
   this directory, and unpack it:

   java-candc$ tar xzvf java-candc-v0.95.tgz

   read the readme file in java-candc for further instructions on how to
   compile and run the parser.

   the id125 parser requires around 6gb of ram.

   this report can also be found in docs/report.pdf.
