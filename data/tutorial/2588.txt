readme.md

graphparser

   graphparser is decribed in the paper
   [108]http://sivareddy.in/papers/reddy2014semanticparsing.pdf

installation

   graphparser is written in java, and requires few external java
   libraries. you can install them using

     ./install.py ungrounded

ungrounded semantic parser

   graphparser can parse natural language sentences to logical parses and
   graphs. run the following command

     cat input.txt | sh run.sh

online demo

   online demo of graphparser can be accessed at
   [109]http://sivareddy.in/graphparser.html

evaluation datasets

   if you are interested in evaluation datasets, and the output of
   graphparser on the test datasets, you can download them using

     ./install.py evaluation

   the datasets will be downloaded to the folders data/tacl_splits and
   data/tacl_ouput.

grounded semantic parser

   to replicate tacl results, you will have to install freebase sparql
   endpoint. please email me personally.
