project description

   theanolm is a recurrent neural network id38 toolkit
   implemented using theano. theano allows the user to customize and
   extend the neural network very conveniently, still generating highly
   efficient code that can utilize multiple gpus or cpus for parallel
   computation. theanolm allows the user to specify arbitrary network
   architecture. new layer types and optimization methods can be easily
   implemented.
