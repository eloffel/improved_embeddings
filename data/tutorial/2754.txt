projects



   _______________________________________________________________________

                [1]maxent

mature java package for training and using maximum id178 models.

   [2]opennlp id35 library

a collection of natural language processing components and tools which
provide support for parsing and realization with combinatory categorial
grammar (id35). this project is an off-shoot of [3]grok.

         [4]opennlp tools

a collection of natural language processing tools which use the maxent
package to resolve ambiguity. the package include a sentence detector,
tokenizer, pos-tagger, shallow and full syntactic parser, and named-entity
detector.

             [5]wordfreak

java-based linguistic annotation tool.



   _______________________________________________________________________

                           [6]agtk

a suite of software components for building tools for annotating linguistic
signals, time-series data which documents any kind of linguistic behavior
(e.g. audio, video). the internal data structures are based on annotation
graphs.

              [7]arithmetic coding

a java package arithmetic coding and ppm (adaptive variable-length > id165
language models for compression)

                      [8]comlintoo

a set of perl tools for computational linguistics (esp. corpus handling and
(permutation) statistics).

   [9]attribute-logic engine (ale)

a freeware logic programming and grammar parsing and generation system

                           [10]edg

a lisp system for developing and displaying hpsg

                       [11]ellogon

an lgpl component-based natural language engineering platform written in c,
c++, java, tcl, perl, and python

                        [12]emdros

a text database engine for analyzed or annotated text.

                      [13]freeling

an open source suite of language analyzers.

                        [14]guitar

a general tool for id2.

                 [15]heart of gold

middleware for combining shallow and deep nlp components.

                           [16]leo

a project to provide an architecture for defining xml specifications of
grammars for different natural language parsing systems and tools for
converting grammars automatically between those systems

                           [17]lkb

the lkb system is a grammar and lexicon development environment for use with
constraint-based linguistic formalisms.

                        [18]mallet

a machine learning for language toolkit written in java

                    [19]minorthird

a collection of java classes for storing text, annotating text, and learning
to extract entities and categorize text.

      [20]ngram statistics package

allows for the counting and measuring of ngrams in text.

                          [21]nltk

a python package intended to simplify the task of programming natural
language systems.

                       [22]nlpfarm

a collection of nlp libraries, tools and demo applications. current focus is
mainly on parsing and dialogue systems.

                   [23]senserelate

implements a id51 algorithm using id138::similarity

                     [24]tiger api

library which allows java programmers to easily access the structure of any
corpus given as a tiger-xml file.

         [25]web as corpus toolkit

a collection of programs that can be used to create a (large) text corpus
from a list of urls.

                          [26]weka

a collection of machine learning algorithms for data mining tasks.

                          [27]weta

the waikato environment for text analysis

           [28]id138::similarity

provides measures of semantic relatedness using id138.
