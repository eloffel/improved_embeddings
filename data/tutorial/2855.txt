admin panels

   libraries for administrative interfaces.
     * [193]ajenti - the admin panel your servers deserve.
     * [194]django-grappelli - a jazzy skin for the django
       admin-interface.
     * [195]django-suit - alternative django admin-interface (free only
       for non-commercial use).
     * [196]django-xadmin - drop-in replacement of django admin comes with
       lots of goodies.
     * [197]flask-admin - simple and extensible administrative interface
       framework for flask.
     * [198]flower - real-time monitor and web admin for celery.
     * [199]wooey - a django app which creates automatic web uis for
       python scripts.

algorithms and design patterns

   python implementation of algorithms and design patterns.
     * [200]algorithms - minimal examples of data structures and
       algorithms in python.
     * [201]pypattyrn - a simple yet effective library for implementing
       common design patterns.
     * [202]python-patterns - a collection of design patterns in python.
     * [203]sortedcontainers - fast, pure-python implementation of
       sortedlist, sorteddict, and sortedset types.

audio

   libraries for manipulating audio and its metadata.
     * audio
          + [204]audioread - cross-library (gstreamer + core audio + mad +
            ffmpeg) audio decoding.
          + [205]dejavu - audio fingerprinting and recognition.
          + [206]mingus - an advanced music theory and notation package
            with midi file and playback support.
          + [207]pyaudioanalysis - python audio analysis library: feature
            extraction, classification, segmentation and applications
          + [208]pydub - manipulate audio with a simple and easy high
            level interface.
          + [209]timeside - open web audio processing framework.
     * metadata
          + [210]beets - a music library manager and [211]musicbrainz
            tagger.
          + [212]eyed3 - a tool for working with audio files, specifically
            mp3 files containing   metadata.
          + [213]mutagen - a python module to handle audio metadata.
          + [214]tinytag - a library for reading music meta data of mp3,
            ogg, flac and wave files.

authentication

   libraries for implementing authentications schemes.
     * oauth
          + [215]authlib - javascript object signing and encryption draft
            implementation.
          + [216]django-allauth - authentication app for django that "just
            works."
          + [217]django-oauth-toolkit - oauth 2 goodies for django.
          + [218]oauthlib - a generic and thorough implementation of the
            oauth request-signing logic.
          + [219]python-oauth2 - a fully tested, abstract interface to
            creating oauth clients and servers.
          + [220]python-social-auth - an easy-to-setup social
            authentication mechanism.
     * jwt
          + [221]pyjwt - json web token implementation in python.
          + [222]python-jose - a jose implementation in python.
          + [223]python-jwt - a module for generating and verifying json
            web tokens.

build tools

   compile software from source code.
     * [224]bitbake - a make-like build tool for embedded linux.
     * [225]buildout - a build system for creating, assembling and
       deploying applications from multiple parts.
     * [226]platformio - a console tool to build code with different
       development platforms.
     * [227]pybuilder - a continuous build tool written in pure python.
     * [228]scons - a software construction tool.

built-in classes enhancement

   libraries for enhancing python built-in classes.
     * [229]dataclasses - (python standard library) data classes.
     * [230]attrs - replacement for __init__, __eq__, __repr__, etc.

     * [231]bidict - efficient, pythonic bidirectional map data structures
       and related functionality..
     * [232]box - python dictionaries with advanced dot notation access.
     * [233]dotteddict - a library that provides a method of accessing
       lists and dicts with a dotted path notation.
