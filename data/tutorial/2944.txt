ipython kernel[13]  

   ipython provides extensions to the python programming language that
   make working interactively convenient and efficient. these extensions
   are implemented in the ipython kernel and are available in all of the
   ipython frontends (notebook, terminal, console and qt console) when
   running this kernel.
