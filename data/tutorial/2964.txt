data science and big data with python

   (c) 2016 [53]steve phelps

   this is a collection of ipython notebooks that i use to teach topics
   relating to data science and big data.
     * [54]programming in python
     * [55]numerical computing
     * [56]relational data
     * [57]analysing structured data using pandas
     * [58]map-reduce programming and apache spark
     * [59]column-oriented databases with hbase and happybase
