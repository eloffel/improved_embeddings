readme.md

   ##statlearning notebooks##

   ipython notebooks that implement the r code for the [66]statlearning:
   statistical learning online course from stanford university taught by
   profs trevor hastie and rob tibshirani. the original code for the
   classes were written in r.

   list of notebooks:
     * [67]chapter 2: basic operations
     * [68]chapter 3: id75
     * [69]chapter 4: classification
     * [70]chapter 5: cross-validation and bootstrap
     * [71]chapter 6: feature selection
     * [72]chapter 7: nonlinear models
     * [73]chapter 8: id90
     * [74]chapter 9: support vector machines
     * [75]chapter 10: unsupervised methods

   the notebooks are also accessible from [76]a gallery of interesting
   ipython notebooks under the statistics, machine learning and data
   science section. more information in [77]my blog post.

   this project was linked from [78]statistics.md of this project
   containing [79]many curated lists of links to python scientific
   programming resources.
