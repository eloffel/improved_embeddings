seaborn: statistical data visualization[17]  

   [scatterplot_matrix_thumb.png?v=0.9.0]

   seaborn is a python data visualization library based on [18]matplotlib.
   it provides a high-level interface for drawing attractive and
   informative statistical graphics.

   for a brief introduction to the ideas behind the library, you can read
   the [19]introductory notes. visit the [20]installation page to see how
   you can download the package. you can browse the [21]example gallery to
   see what you can do with seaborn, and then check out the [22]tutorial
   and [23]api reference to find out how.

   to see the code or report a bug, please visit the [24]github
   repository. general support issues are most at home on
   [25]stackoverflow, where there is a seaborn tag.
