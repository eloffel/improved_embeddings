essential cheat sheets for machine learning and deep learning engineers

   go to the profile of kailash ahirwar
   [15]kailash ahirwar (button) blockedunblock (button) followfollowing
   may 28, 2017

   machine learning is complex. for newbies, starting to learn machine
   learning can be painful if they don   t have right resources to learn
   from. most of the machine learning libraries are difficult to
   understand and learning curve can be a bit frustrating. i am creating a
   repository on github([16]cheatsheets-ai) containing cheatsheets for
   different machine learning frameworks, gathered from different sources.
   do visit the github repository, also, contribute cheat sheets if you
   have any. thanks.
