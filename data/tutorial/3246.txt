readme.md

deepnet

implementation of some deep learning algorithms.

   gpu-based python implementation of
    1. feed-forward neural nets
    2. restricted id82s
    3. deep belief nets
    4. autoencoders
    5. deep id82s
    6. convolutional neural nets

   built on top of the [83]cudamat library by vlad mnih and
   [84]cuda-convnet library by alex krizhevsky.
