readme.md

theano-tutorials

   bare bones introduction to machine learning from id75 to
   convolutional neural networks using theano.

   dataset it's worth noting that this library assumes that the reader has
   access to the mnist dataset. this dataset is freely available and is
   accessible through yann lecun's [82]personal website.

   if you want to automate the download of the dataset, there is an
   included file that will do this for you. simply run the following: sudo
   ./download_mnist.sh

   known issues library not loaded:
   /usr/local/opt/openssl/lib/libssl.1.0.0.dylib this results from a
   broken openssl installation on mac. it can be fixed by uninstalling and
   reinstalling openssl: sudo brew remove openssl brew install openssl
