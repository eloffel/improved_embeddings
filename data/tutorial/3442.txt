uwtime

   uwtime is a temporal semantic parser that detects and resolves time
   expressions in text. the system is described in:
   [1]context-dependent id29 for time expressions
   [2]kenton lee, [3]yoav artzi, [4]jesse dodge and [5]luke zettlemoyer
   in proceedings of the conference of the association for computational
   linguistics (acl), 2014.

   the system was implemented using [6]the university of washington
   id29 framework. the code for replicating the results can be
   found at [7]here. a minimal, standalone version of uwtime designed for
   downstream applications can be found [8]here. you can also try our demo
   below!

   for questions and inquiries please contact [9]kenton lee.
