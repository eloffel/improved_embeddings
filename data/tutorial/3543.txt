readme.md

generative models

   collection of generative models, e.g. gan, vae in pytorch and
   tensorflow. also present here are rbm and helmholtz machine.

note:

   generated samples will be stored in gan/{gan_model}/out (or
   vae/{vae_model}/out, etc) directory during training.

what's in it?

generative adversarial nets (gan)

    1. [92]vanilla gan
    2. [93]conditional gan
    3. [94]infogan
    4. [95]wasserstein gan
    5. [96]mode regularized gan
    6. [97]coupled gan
    7. [98]auxiliary classifier gan
    8. [99]least squares gan
    9. [100]boundary seeking gan
   10. [101]energy based gan
   11. [102]f-gan
   12. [103]generative adversarial parallelization
   13. [104]discogan
   14. [105]adversarial id171 & [106]adversarially learned
       id136
   15. [107]boundary equilibrium gan
   16. [108]improved training for wasserstein gan
   17. [109]dualgan
   18. [110]magan: margin adaptation for gan
   19. [111]softmax gan
   20. [112]gibbsnet

variational autoencoder (vae)

    1. [113]vanilla vae
    2. [114]conditional vae
    3. [115]denoising vae
    4. [116]adversarial autoencoder
    5. [117]adversarial id58

restricted id82 (rbm)

    1. [118]binary rbm with contrastive divergence
    2. [119]binary rbm with persistent contrastive divergence

helmholtz machine

    1. [120]binary helmholtz machine with wake-sleep algorithm

dependencies

    1. install miniconda [121]http://conda.pydata.org/miniconda.html
    2. do conda env create
    3. enter the env source activate generative-models
    4. install [122]tensorflow
    5. install [123]pytorch
