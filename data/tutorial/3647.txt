   a mooc on id76, [4]cvx101, was run from 1/21/14 to
   3/14/14. if you register for it, you can access all the course
   materials.

   more material can be found at the web sites for [5]ee364a (stanford) or
   [6]ee236b (ucla), and our own web pages. source code for almost all
   examples and figures in part 2 of the book is available in [7]cvx (in
   the [8]examples directory), in [9]cvxopt (in the book examples
   directory), and in [10]cvxpy. source code for examples in chapters 9,
   10, and 11 can be found [11]here. instructors can obtain complete
   solutions to exercises by email request to us; please give us the url
   of the course you are teaching.

   if you find an error not listed in our [12]errata list, please do let
   us know about it.
