keras resources

   this is a directory of tutorials and open-source code repositories for
   working with keras, the python deep learning library.

   if you have a high-quality tutorial or project to add, please open a
   pr.

     * [67]keras.io - keras documentation

     * [172]keras-vis: neural network visualization toolkit for keras.

projects built with keras

     * [173]rocalphago: an independent, student-led replication of
       deepmind's 2016 nature publication, "mastering the game of go with
       deep neural networks and tree search"
     * [174]betago: deep learning go bots using keras
     * [175]deepjazz: deep learning driven jazz generation using keras
     * [176]dataset-sts: semantic text similarity dataset hub
     * [177]snli-entailment: independent implementation of attention model
       for id123 from the paper [178]"reasoning about
       entailment with neural attention".
     * [179]headline generator: independent implementation of
       [180]generating news headlines with recurrent neural networks
     * [181]lipnet: independent implementation of [182]lipnet: end-to-end
       sentence-level lipreading.
     * [183]caption_generator: an implementation of image caption
       generation in natural language inspired from [184]show and tell: a
       neural image caption generator.
     * [185]id4-keras: id4 using keras.
     * [186]conx - easy-to-use layer on top of keras, with visualizations
       (eg, no knowledge of numpy needed)
