readme.md

tensorflow tutorial

   welcome to the tensorflow tutorial.

   please install tensorflow on your computer before you arrive.

   for this tutorial, you can do one of the following:
    1. [71]install tensorflow directly on your mac. this is recommended,
       as it installs the least amount on your computer and will run the
       fastest.
    2. [72]use docker on mac, running tensorflow in a local vm. if for
       some reason you can't install tensorflow natively or already have
       docker installed, you can try this.
    3. [73]install tensorflow directly on your ubuntu/linux machine.

   note that tensorflow is not currently supported natively on windows.

   if you run into trouble, our full install instrictions are available on
   [74]http://tensorflow.org.
