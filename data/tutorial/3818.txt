readme.md

                  [80][pytorch_logo_2018.svg?sanitize=true]
     __________________________________________________________________

   this repository provides tutorial code for deep learning researchers to
   learn [81]pytorch. in the tutorial, most of the models were implemented
   with less than 30 lines of code. before starting this tutorial, it is
   recommended to finish [82]official pytorch tutorial.

table of contents

1. basics

     * [83]pytorch basics
     * [84]id75
     * [85]id28
     * [86]feedforward neural network

2. intermediate

     * [87]convolutional neural network
     * [88]deep residual network
     * [89]recurrent neural network
     * [90]bidirectional recurrent neural network
     * [91]language model (id56-lm)

3. advanced

     * [92]id3
     * [93]variational auto-encoder
     * [94]neural style transfer
     * [95]image captioning (id98-id56)

4. utilities

     * [96]tensorboard in pytorch

getting started

$ git clone https://github.com/yunjey/pytorch-tutorial.git
$ cd pytorch-tutorial/tutorials/path_to_project
$ python main.py

     * [97]python 2.7 or 3.5+
