readme.md

cs1001.py

   recitation notebooks for the course [109]extended introduction to
   computer science with python, given at tel-aviv university in spring
   2013. these recitations were given by [110]yoav ram. the course is
   given as a first cs course to undergraduate cs students on the first or
   second semester of the first year of their studies towards a bsc. in
   computer science.
     * [111]recitation 1: variables, operators, flow control
     * [112]recitation 2: input, collatz conjecture, lists, functions
     * [113]recitation 3: diviors, timing operations, binary system, base
       conversion, python's memory model
     * [114]recitation 4: time complexity, big o notation, primality
       testing, diffie-hellman key exchange
     * [115]recitation 5: object-oriented programming, recursion
     * [116]recitation 6: recursion, quicksort
     * [117]recitation 7: lambda expression, high-order functions, matrix
       class, file handling
     * [118]recitation 8: hash functions, hash tables, finding repeating
       substrings
     * [119]recitation 9: linked list, iterators, generators
     * [120]recitation 10: rabin-karp algorithm for string matching
     * [121]recitation 11: huffman code
     * [122]recitation 12: lz compression
     * [123]recitation 13: image processing, denoising algorithms
     * [124]recitation 14: error detection and correction

technical details

   the recitation notebooks are written with [125]ipython notebook 0.13
   and python 3.2. some of the notebooks require external packages like
   numpy and matplotlib.
