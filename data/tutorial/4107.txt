readme.md

neural caption generator with attention

     * tensorflow implementation of "show, attend and tell"
       [80]http://arxiv.org/abs/1502.03044
     * borrowed most of the idea from the author's source code
       [81]https://github.com/kelvinxu/arctic-captions

code

     * make_flickr_dataset.py: extracts conv5_3 layer activations of vgg
       network for flickr30k images, and save them in 'data/feats.npy'
     * model_tensorflow.py: main codes

usage

     * download flickr30k dataset.
     * extract vgg conv5_3 features using make_flickr_dataset.py
     * train: run train() in model_tensorflow.py
     * test: run test() in model_tensorflow.py
