   in this machine learning tutorial we delve into another method of
   preprocessing, which deals with the scaling of the data.
   more specifically, we're going to apply a minmaxscaler on a dataset.
   this type of scaler brings each feature into a value range of 0 to 1,
   which is really important for the correct training of some machine
   learning algorithms, like for example support vector machines or neural
   networks.
