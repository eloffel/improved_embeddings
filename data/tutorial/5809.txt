readme.md

deeplearning4j examples

   repository of deeplearning4j neural net examples:
     * mlp neural nets
     * convolutional neural nets
     * recurrent neural nets
     * tsne
     * id97 & glove
     * anomaly detection
     __________________________________________________________________

build and run

   use [84]maven to build the examples.
mvn clean package

   run the runexamples.sh script to run the examples (requires [85]bash).
   it will list the examples and prompt you for the one to run. pass the
   --all argument to run all of them. (other options are shown with -h).
./runexamples.sh [-h | --help]

documentation

   for more information, check out [86]deeplearning4j.org and its
   [87]javadoc.

   if you notice issues, please log them, and if you want to contribute,
   submit a pull request. input is welcome here.
