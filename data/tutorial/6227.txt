awesome community-curated nlp list

   to contribute: this list is community curated, anyone can do a
   pull-request to add to the list. and it will be merged once 5 person
   have verified that the pr is not spam.

speech nlp

     * automatic id103 (speech-to-text)
          + [64]kaldi
          + [65]cmu sphinx
          + [66]julius
          + [67]rwth asr
     * id133 (text-to-speech)
          + [68]merlin
          + [69]festival
          + [70]hts
          + [71]espeak
          + [72]covarep
          + [73]free tts
          + [74]ekho
     * generic speech analysis/modelling tools
          + [75]hidden markov model toolkit
          + [76]praat

            izers

            oftware

language specific text nlp suites

     * arabic
          + [95]safar: software architecture for arabic language
            processing
     * cantonese
          + [96]pycantonese: cantonese linguistics and nlp in python
     * chinese
          + [97]snownlp: simplified chinese text processing
     * persian
          + [98]hazm: python library for digesting persian text.
     * dutch
          + [99]frog: an advanced nlp suite for dutch
     * italian
          + [100]tint: lend color to your italian texts!
     * korean
          + [101]konlpy: korean nlp in python

pre-processing (id121 / id30 / id52 / etc.)

     * ngrams
          + [102]colibri core - c++ and python tools for id165s and
            skipgrams
     * id30
          + [103]snowball stemmers
          + [104]serbianstemmer
          + [105]whoosh stemmers
     * tokenizers
          + [106]elephant: sequence labeling for word and sentence
            segmentation
          + [107]toktok: a fast, simple, multilingual tokenizer
          + [108]ucto: an advanced rule-based unicode-aware tokenizer

deep linguistic processing

   the deep here isn't "deep learing" deep ;p , see
   [109]https://en.wikipedia.org/wiki/deep_linguistic_processing

          + [110]delph-in: deep linguistic processing with hpsg
          + [111]english resource grammar
     * id35 (id35)
          + [112]id352pst : a tool for converting id35 derivations into
            ptb-style phrase structure trees
