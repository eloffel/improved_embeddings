readme.md

id195 chatbot

   this is a 200 lines implementation of twitter/cornell-movie chatbot,
   please read the following references before you read the code:
     * [87]practical-id195
     * [88]the unreasonable effectiveness of recurrent neural networks
     * [89]understanding id137 (optional)

prerequisites

     * python
     * [90]tensorflow >= 1.2
     * [91]tensorlayer >= 1.6.3
