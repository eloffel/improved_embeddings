readme.md

deep semantic similarity model

   my keras implementation of the deep semantic similarity model
   (dssm)/convolutional latent semantic model (clsm) described [76]here.
   as search data sets are generally proprietary, you will have to provide
   your own data to use with the code.

additional references

    1. [77]http://research.microsoft.com/pubs/238873/wsdm2015.v3.pdf -
       slides giving a high level overview of the dssm and how it can be
       used for information retrieval.
    2. [78]http://research.microsoft.com/en-us/projects/dssm/ - microsoft
       research's summary of the dssm (includes many more references).
