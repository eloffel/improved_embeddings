readme.md

neural network from scratch

     * implementation of a simple id158 from scratch
       in python.
     * classification of mnist dataset.

   accompanying blog posts:

   dl01:
   [79]https://hackernoon.com/dl01-writing-a-neural-network-from-scratch-t
   heory-c02ccc897864

   dl02:
   [80]https://hackernoon.com/dl02-writing-a-neural-network-from-scratch-c
   ode-b32f4877c257

getting started

   git clone
   git@github.com:thesemicolonguy/neural-network-from-scratch.git

prerequisites

   python3: sudo apt-get update && sudo apt-get install python3.6

   numpy: pip3 install numpy

   dill: pip3 install dill

   python-mnist: pip3 install python-mnist

running the classifier

   python3 classify-mnist.py
