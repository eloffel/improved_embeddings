uriel typological database

introduction

   the uriel knowledge base is a structured compendium of information on
   language typology and language universals that is being developed as
   part of darpa's lorelei project.

   for the most recent version of uriel and lang2vec, use pip to install
   the [1]lang2vec package from pypi.

   download the newest non-pypi release [2]here. read the readme in
   [3]markdown.

releases

     * [4]uriel v0.0

     * initial release.

     [5]uriel v0.1
     * covers more features and languages.
     * provides more data points.
     * corrects several crucial bugs.

     [6]uriel v0.2
     * covers more features and languages.
     * provides more data points.
     * introduces additional data format (.npz) for improved performance.
     * introduces    mini-grammars    derived from typological features.

     [7]uriel v0.3.0
     * treatment of macrolanguages. macrolanguages like arabic are now
       included and their feature values assigned in a principled fashion.
     * improvements in phylogenetic identifications and geolocations.
       these data, located in the geodata directory, are much improved.
     * improved feature prediction. improvement in the accuracy of feature
       prediction from 92% to 93%.
     * mini-inversion grammars. inversion grammars for language pairs in
       the uriel knowledge bank now available on request.

   information on unpacking .tar.xz archives is available [8]here.

     [9]uriel lang2vec
     * a tool for querying the uriel database.
