unit: unsupervised image-to-image translation networks

license

   copyright (c) 2018 nvidia corporation. all rights reserved. licensed
   under the cc by-nc-sa 4.0 license
   ([102]https://creativecommons.org/licenses/by-nc-sa/4.0/legalcode).

code usage

     * please check out our [103]tutorial.
     * for multimodal (or many-to-many) image translation, please check
       out our new work on [104]munit.

what's new.

     * 05-02-2018: we now adapt [105]munit code structure. for reproducing
       experiment results in the nips paper, please check out
       [106]version_02 branch.
     * 12-21-2017: release pre-trained synthia-to-cityscape image
       translation model. see [107]usage.md for usage examples.
     * 12-14-2017: added multi-scale discriminators described in the
       [108]pix2pixhd paper. to use it simply make the name of the
       discriminator cocomsdis.

paper

   [109]ming-yu liu, thomas breuel, jan kautz, "unsupervised
   image-to-image translation networks" nips 2017 spotlight,
   arxiv:1703.00848 2017
