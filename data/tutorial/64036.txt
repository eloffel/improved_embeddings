readme.md

   a id4 model written in pytorch.

   for a up-to-date pytorch implementation of basic vanilla attentional
   id4, please refer to [83]this repo

   with 256-dimensional lstm hidden size, it achieves a training speed of
   14000 words/sec and 26.9 id7 score on the iwslt 2014 germen-english
   dataset (ranzato et al., 2015).

file structure

     * id4.py: main file
     * vocab.py: script used to generate .bin vocabulary file from
       parallel corpus
     * util.py: script containing helper functions
     * run_raml_exp.py|test_raml_models.py: helper scripts to train|test
       raml models with different temperature settings (refer to [norouzi
       et al., 2016] for details)

usage

     * generate vocabulary files

python vocab.py

     * vanilla maximum likelihood training

. scripts/run_id113.sh

     * reward augmented maximum likelihood training (norouzi et al., 2016)

. scripts/run_raml.sh

     * id23 (coming soon)
