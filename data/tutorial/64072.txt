stanford-tensorflow-tutorials

   this repository contains code examples for the course cs 20: tensorflow
   for deep learning research.
   it will be updated as the class progresses.
   detailed syllabus and lecture notes can be found [93]here.
   for this course, i use python3.6 and tensorflow 1.4.1.

   for the code and notes of the previous year's course, please see the
   folder 2017 and the website
   [94]https://web.stanford.edu/class/cs20si/2017

   for setup instruction and the list of dependencies, please see the
   setup folder of this repository.
