ai2 reasoning challenge (arc)

   think you have solved id53? try the ai2 reasoning
   challenge (arc)! the arc dataset contains 7,787 genuine grade-school
   level, multiple-choice science questions, assembled to encourage
   research in advanced question-answering. the dataset is partitioned
   into a challenge set and an easy set, where the former contains only
   questions answered incorrectly by both a retrieval-based algorithm and
   a word co-occurrence algorithm. the questions are partitioned as
   follows:
     * challenge train: 1,119
     * challenge dev: 299
     * challenge test: 1,172
     * easy train: 2,251
     * easy dev: 570
     * easy test: 2,376

   for more information about this dataset please read: [16]think you have
   solved id53? try arc, the ai2 reasoning challenge.

   to join in on the discussion about this dataset, please visit the
   [17]arc discussion group.
