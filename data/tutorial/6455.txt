1. tools

   notations:
     *      easy to install and use
     *      commercial-friendly license
     *      pretrained models are available or not needed

word id121, sentence splitting

     * [92]huntoken              hungarian word and sentence splitter
     * [93]quntoken              new hungarian tokenizer based on quex, huntoken

morphology

     * [94]emmorph (humor)      hungarian morphological analyzer based on
       humor
     * [95]emmorphpy         a wrapper, a lemmatizer and rest api implemented in
       python for emmorph (humor) hungarian morphological analyzer
     * [96]hunmorph          is an open source tool and programming library for
       spell-checking, id30 and morphological analysing of
       agglutinative, german and other languages.
     * [97]hunmorph-foma          hungarian morpholical analyzer and generator
       based on hunmorph.
     * [98]hunspell              is an open-source spell-checker, stemmer and
       morphological analyzer
     * [99]lara-hungarian-nlp              lara is a lightweight python nlp library
       for chatbots in hungarian.
     * [100]lemmagen              project aims at providing standardized open
       source multilingual platform for lemmatisation. ([101]python
       package for v2 | [102]c# project for v3)

pos / morphological taggers

     * [103]hunpos              hunpos is an open source reimplementation of tnt,
       the well known part-of-speech tagger by thorsten brants.
     * [104]purepos          open source morphological tagger based on hunpos
     * [105]purepos.py          python wrapper for purepos

taggers / chunkers

     * [106]huntag          a sequential tagger for nlp using maximum id178
       learning and id48
     * [107]huntag3          improved version of the original huntag
     * [108]szegedner              id39 tool for hungarian and
       english
     * [109]dbpedia spotlight              dbpedia spotlight is a tool for
       automatically annotating mentions of dbpedia resources in text.
       [110]docker image

pipelines with hungarian nlp components

     * [111]magyarlanc          a toolkit for the basic linguistic processing of
       hungarian
     * [112]magyarlanc_spark          spark wrapper for magyarlanc
     * [113]spacy              industrial-strength natural language processing
       (nlp) with python and cython ([114]hungarian models)
     * [115]hunlp          unified java and rest api for magyarlanc and
       szegedner
     * [116]hunlp-gate      gate plugin containing hungarian nlp tools as
       gate processing resources
     * [117]trendminer hungarian processing pipeline      hungarian nlp
       pipeline for social media text analysis (trendminer project)
     * [118]google syntaxnet          neural models of syntax
     * [119]udpipe              is a trainable pipeline for id121, tagging,
       lemmatization and id33 of conll-u files
     * [120]polyglot              is a natural language pipeline that supports
       massive multilingual applications.
     * [121]emtsv          is a text processing system with inter-module
       communication via tsv + rest api
     * [122]stanfordnlp          is a python nlp library for many human
       languages including hungarian
     * [123]spacy stanfordnlp          wraps the stanfordnlp library, so you can
       use stanford's models as a spacy pipeline

syntactic parsers

     * [124]hunpars          a rule based hungarian syntactical analyzer
     * [125]hunparse          an nltk-based parser using kr-style morphological
       annotation
     * [126]anagramma parser a parser based on psycholinguistics
       principles
     * [127]benepar a high-accuracy parser with models for 11 languages,
       implemented in python. based on constituency parsing with a
       self-attentive encoder from acl 2018.

semantic analysis

     * [128]sentimentanalysishun              is an open-source id31
       tool for hungarian language, written in python.

other

     * [129]emlam              preprocessing scripts for hungarian language
       modeling
     * [130]pywnxml              python3 api for id138 xml (hungarian id138 /
       balkanet / visdic format)

2. datasets

corpora

     * [131]hungarian webcorpus with over 1.48 billion words unfiltered
       (589m words fully filtered), this is by far the largest hungarian
       language corpus, and unlike the hungarian national corpus (125m
       words), it is available in its entirety under a permissive open
       content license.
     * [132]emlam a id38 benchmark corpus for hungarian,
       similar to the one billion word corpus (chelba, 2014) for english.
     * [133]leipzig corpora contains randomly selected sentences in the
       language of the corpus and are available in sizes from 10,000
       sentences up to 1 million sentences. the sources are either
       newspaper texts or texts randomly collected from the web.
     * [134]web2corpus automatically create multilingual web corpus
     * [135]conll 2017: automatically annotated raw texts and word
       embeddings automatic segmentation, id121 and morphological
       and syntactic annotations of raw texts in 45 languages, generated
       by [136]udpipe, together with id27s of dimension 100
       computed from lowercased texts by [137]id97
     * [138]opinhubank opinhubank is a human-annotated corpus to aid the
       research of opinion mining and id31 in hungarian
     * [139]the hungarian forum corpus for opinion mining this database is
       the first one dedicated to opinion mining in hungarian. the data
       for further processing were gathered from the posts of the forum
       topic of the hungarian government portal dealing with the
       referendum about dual citizenship.
     * [140]szeged treebank the szeged treebank is the largest fully
       manually annotated treebank of the hungarian language
     * [141]szeged dependency treebank the szeged dependency treebank is a
       dependency-tree format version of the szeged treebank.
     * [142]universal dependencies
     * [143]hungarian named entity corpora the named entity corpus for
       hungarian is a subcorpus of the szeged treebank, which contains
       full syntactic annotations done manually by linguist experts.
     * [144]hunnerwiki a silver standard corpus for hungarian named entity
       recognition
     * [145]mazsola database containes 28m sentences from the mnsz1 corpus
       annotated with shallow syntactic analysis
     * [146]hungarian word sense disambiguated corpus containing 39
       suitable word form samples for the purpose of word sense
       disambiguation
     * [147]hunlearner is a learners' corpus of hungarian containing
       written data from 35 students majoring in hungarian studies at the
       university of zagreb, croatia. texts were morphologically and
       syntactically analyzed by the magyarlanc tool.
     * [148]hunglish corpus the hunglish corpus is a free sentence-aligned
       hungarian-english parallel corpus of about 120 million words in 4
       million sentence pairs.
     * [149]szegedparallel the english-hungarian parallel corpus contains
       texts selected on the basis of grammatical and translational
       criteria.
     * [150]hunor a hungarian-russian parallel corpus comprises
       approximately 800 thousand words.
     * [151]conll 2017 shared task hungarian data automatic segmentation,
       id121 and morphological and syntactic annotations of raw
       texts from the common crawl
     * [152]css10 a collection of single speaker speech datasets for 10
       languages including hungarian

word vectors

     * [153]fastext wikipedia pre-trained word vectors for 90 languages,
       trained on wikipedia using fasttext.
     * [154]fastext common crawl & wikipedia pre-trained word vectors for
       157 languages, trained on wikipedia and the common crawl using
       fasttext's cbow model.
     * [155]fasttext_multilingual id73 vectors in 78
       languages
     * [156]polyglot vectors polgyglot embeddings on wikipedia
     * [157]wordvectors pre-trained id97 and fasttext word vectors on
       wikipedia of 30+ languages
     * [158]hunembed0.0 a id97 id27 trained on the
       concatenation of the hungarian webcorpus and the hungarian national
       corpus in 600 dimensions with a cut-off of 10 words.
     * [159]szeged word vectors id27s (id97 & fasttext) for
       hungarian trained on 4.3 billion tokens
     * [160]questions-words-hu hungarian analogical questions following
       mikolov et al.
     * [161]conceptnet numberbatch conceptnet numbermatch multi- and
       cross-lingual semantic id27s
     * [162]multi-sense id27s
     * [163]elmo representations deep contextualized word representation
       trained for many languages

linguistic resources

     * [164]morphdb.hu is an open source morphological database of
       hungarian, consisting of a lexicon and morphological grammar that
       are based on well-founded theoretical decisions.
     * [165]huwn hungarian id138
     * [166]hungarian sentiment lexicon the dictionaries were manually
       created on the basis of id138-affect lexicons.
     * [167]4lang concept dictionary using eilenberg machines
     * [168]named entity lists for hungarian
     * [169]mazsola isz lists 500k verb frames extracted from the mazsola
       database
     * [170]manocska merges verb frames existing databases
     * [171]prevlex list of phrasel verbs
     * [172]panmorph tagsets and description of hungarian morphological
       analysers.

linked open data

     * [173]wikipedia dumps
     * [174]dbpedia dumps
     * [175]huwn.rdf hungarian id138 in rdf format for the linked open
       data cloud
     * [176]conceptnet an open, multilingual id13 (with partial
       hungarian support)
