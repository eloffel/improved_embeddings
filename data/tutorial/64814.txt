readme.md

tree-structured id137

   a [95]pytorch based implementation of tree-lstm from kai sheng tai's
   paper [96]improved semantic representations from tree-structured long
   short-term memory networks.

requirements

     * [97]pytorch deep learning library
     * [98]tqdm: display progress bar
     * [99]meowlogtool: a logger that write everything on console to file
     * java >= 8 (for stanford corenlp utilities)
     * python >= 3

usage

   first run the script ./fetch_and_preprocess.sh

   this downloads the following data:
     * [100]stanford sentiment treebank (sentiment classification task)
     * [101]glove word vectors (common crawl 840b) -- warning: this is a
       2gb download!

   and the following libraries:
     * [102]stanford parser
     * [103]stanford pos tagger

sentiment classification

python sentiment.py --name <name_of_log_file> --model_name <constituency|depende
ncy> --epochs 10

   we have not fully test on fine grain classification yet. binary
   classification accuracy on both model are the same in original paper.
