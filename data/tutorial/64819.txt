generate internal query parse tree in xml for further processing

   sometimes xml output of sql query tree is a starting point for doing
   further processing of sql script. with the help of this demo you can
   generate xml output of sql script quickly. you can also modify this
   demo to create customized xml output to meet your own standards.

   this demo also illustrates how to get ast of sql queries using apis
   provided by general sql parser library.

   1. [14]try online demo

   2. download this demo:
   java version: download general sql parser java version from [15]this
   page, check toxml.java under demos/visitors/ directory.

   .net version: download general sql parser .net version from [16]this
   page, check demo under democollections\xmloutput\ directory.

   3. [17]xml schema which define the legal building blocks of an xml
   document generated by this tool
