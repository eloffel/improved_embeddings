readme.md

colorless green id56s

   this repository contains data and evaluation code for the following
   paper:

   k. gulordava, p. bojanowski, e. grave, t. linzen, m. baroni. 2018.
   [81]colorless green recurrent networks dream hierarchically.
   proceedings of naacl. [82][bib] [83][aclweb]

   supplementary materials available [84]here.

   please cite the paper if you use the resources in the repository.

   we expect that most users will be interested in the [85]data directory,
   that contains our training corpora, pre-trained models and agreement
   test sets.
