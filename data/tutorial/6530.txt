machine learning

resources

     * [108]awesome machine learning comprehensive list of machine
       learning resources
     * [109]dive into machine learning collections of links and notebooks
       for a gentle introduction to machine learning
     * [110]topdeeplearning is a list of popular github projects related
       to deep learning (ranked by stars)
     * [111]probabilistic programming and bayesian methods for hackers an
       introduction to bayesian methods + probabilistic programming with a
       computation/understanding-first, mathematics-second point of view.
       all in pure python
     * [112]data science ipython notebooks
     * [113]python data-science handbook
     * [114]deep learning papers reading roadmap

frameworks

     * [115]theano is a python library that allows you to define,
       optimize, and evaluate mathematical expressions involving
       multi-dimensional arrays efficiently
     * [116]tensorflow library for numerical computation using data flow
       graphs. nodes in the graph represent mathematical operations, while
       the graph edges represent the multidimensional data arrays
       (tensors) communicated between them.
     * [117]keras deep learning library for [118]theano, [119]tensorflow
       and [120]cntk.
     * [121]caffe deep learning framework made with expression, speed, and
       modularity in mind. written in c++ and has python bindings.
     * [122]torch provides several tools for fast tensor mathematics,
       storage interfaces and machine learning models. written in c with
       lua interface.
     * [123]pytorch tensors and dynamic neural networks in python with
       strong gpu acceleration
     * [124]vowpal wabbit is a machine learning system which pushes the
       frontier of machine learning with techniques such as online,
       hashing, allreduce, reductions, learning2search, active, and
       interactive learning. writtent in c++ with bindings for python and
       other languages.
     * [125]scikit learn is a python module for machine learning built on
       top of [126]scipy
     * [127]cntk computational network toolkit. a c++ library by microsoft
       research.
     * [128]xgboost an optimized distributed gradient boosting library
       designed to be highly efficient, flexible and portable. written in
       c++ with python integration.
     * [129]tpot is a python tool that automatically creates and optimizes
       machine learning pipelines using genetic programming.

neural networks

     * [130]brainforge a neural networking library based on numpy only
     * [131]deeplearn.js a neural network library for the web
     * [132]opennn a neural network c++ library

id23

     * [133]keras-rl deep id23 for keras.
     * [134]gym a toolkit for developing and comparing reinforcement
       learning algorithms. written in python.
     * [135]tflearn is a deep learning library featuring a higher-level
       api for [136]tensorflow.
     * [137]tensorforce a tensorflow library for applied reinforcement
       learning

examples

     * [138]aima python python code for the book [139]artificial
       intelligence: a modern approach
     * [140]tensorflow examples a [141]tensorflow tutorial with popular
       machine learning algorithms implementation

nlp

   natural language processing benefits from [142]recurrent neural network
   algorithms.

resources

     * [143]oxford deep nlp 2017 course lecture slides and course
       description for the deep natural language processing course

analysis

     * [144]natural language toolkit (nltk) is a suite of python modules,
       data sets and tutorials supporting research and development in
       [145]nlp. some of its modules are out of date but still a useful
       resource nonetheless.
     * [146]spacy is a powerful, production ready, nlp library for python
     * [147]fasttext a c++ library for sentence classification
     * [148]textblob is a python library for processing textual data. it
       provides a simple api for diving into common [149]nlp tasks such as
       part-of-speech tagging, noun phrase extraction, id31,
       classification, translation, and more.
     * [150]simhash a python implementation of [151]simhash algorithm for
       detecting near-duplicate web documents
     * [152]langdetect is a port of google's language-detection library to
       python.

tools

     * [153]inflect.py correctly generate plurals, ordinals, indefinite
       articles; convert numbers to words

images

resources

     * [154]convolutional neural network in machine learning, a
       convolutional neural network (id98, or convnet) is a class of deep,
       feed-forward id158 that have successfully been
       applied to analyzing visual imagery.

frameworks

     * [155]tesseract-ocr well tested [156]ocr engine written in c++
     * [157]opencv id161 and machine learning software library.
       the library has more than 2500 optimized algorithms, which includes
       a comprehensive set of both classic and state-of-the-art computer
       vision and machine learning algorithms. these algorithms can be
       used to detect and recognize faces, identify objects, classify
       human actions in videos, track camera movements, track moving
       objects, extract 3d models of objects, produce 3d point clouds from
       stereo cameras, stitch images together to produce a high resolution
       image of an entire scene, find similar images from an image
       database, remove red eyes from images taken using flash, follow eye
       movements, recognize scenery and establish markers to overlay it
       with augmented reality, etc. written in c++ with bindins for most
       languages including python.
     * [158]simplecv is a framework for machine vision, using [159]opencv
       and python. it provides a concise, readable interface for cameras,
       image manipulation, feature extraction, and format conversion.
     * [160]match makes it easy to search for images that look similar to
       each other
     * [161]noteshrink convert scans of handwritten notes to beautiful,
       compact pdfs
     * [162]srez image super-resolution through deep learning
     * [163]covnetjs train convolutional neural networks (or ordinary
       ones) in the browser
