readme.md

python-topic-model

   implementations of various topic models written in python. note that
   some of the implementations (the models with mcmc) are extremely slow.
   i do not recommend to use it for large scale datasets.

current implementations

     * id44
          + [81]collapsed id150
          + [82]variational id136
     * collaborative topic model
          + variational id136
     * relational topic model (vi)
          + [83]exponential link function
     * [84]author-topic model
     * [85]id48-lda
     * discrete infinite logistic normal (diln)
          + variational id136
     * supervised topic model
          + [86]stochastic (gibbs) em

          + collapsed id150
