pomdp example domains

   tony's pomdp file repository page

   this page contains links to download various pomdp parameter files that
   have appeared in the literature. a description of the format of these
   files can be found on the [3]pomdp file format page, with the
   corrresponding more formal [4]pomdp file grammar page

other problems

1d.noisy.pomdp

   noisy transition version of littman's 1d maze

concert.pomdp

   michael littman's concert problem

tiger.95.pomdp

   version of aaai-94 tiger problem with 0.95 discount factor (instead of
   0.75)

stand-tiger.95.pomdp

   slight extension of the tiger problem

line4-2goals.pomdp

   a line of uniquely appearing states with a goal at both ends

4x5x2.95.pomdp.gz

   a 2 floor 4x5 navigation problem

bulkhead.a.pomdp.gz

   non-destructive testing problem of a titanium aircraft engine bulkhead

marking.pomdp

   one of two versions of a problem proposed by tom dean about a robot
   that can mark its environment.

marking2.pomdp

   one of two versions of a problem proposed by tom dean about a robot
   that can mark its environment.

mcc-example1.pomdp

   variations of an example of choosing advertisements for web pages based
   on user actions (made for mcc job talk)

mcc-example2.pomdp

   variations of an example of choosing advertisements for web pages based
   on user actions (made for mcc job talk)

web-ad.pomdp

   variations of an example of choosing advertisements for web pages based
   on user actions (made for mcc job talk)

web-mall-example1.pomdp

   variations of an example of choosing advertisements for web pages based
   on user actions (made for mcc job talk)

saci-s12-a6-z5.95.pomdp.gz

   aircraft identification problem with 12 states, 6 actions and 5
   observations

saci-s100-a10-z31.pomdp.gz

   aircraft identification problem with 100 states, 10 actions and 31
   observations

query.s2.pomdp.gz

   problems for distributed querying decisions with 2 resources

query.s3.pomdp.gz

   problems for distributed querying decisions with 3 4 resources

query.s4.pomdp.gz

   problems for distributed querying decisions with 4 resources

   problem of teaching policies for trying to learn 2 concepts

   problem of teaching policies for trying to learn 3 concepts

problems from anthony cassandra's ph. d. thesis

toy-pomdp-probs.tar.gz

   suite of 9 small problems

hallway.pomdp.gz

   hallway (57 state navigation problem

hallway2.pomdp.gz

   hallway-2 (89 state navigation problem

cit.pomdp.gz

   larger robot navigation problem. a coarse model of the fourth floor of
   the cit building, brown university

mit.pomdp.gz

   larger robot navigation problem. simplified model of the 8th floor of
   mit's computer science building

sunysb.pomdp.gz

   larger robot navigation problem. first floor of computer science
   building, suny at stony brook

pentagon.pomdp.gz

   larger robot navigation problem. inspired by the pentagon building with
   an inner and outer hallway

fourth.pomdp.gz

   larger robot navigation problem. a detailed portion of the four floor,
   cit building, brown university.

iff.pomdp.gz

   approaching aircraft identification problem

machine.pomdp.gz

   model of maintenance for a machine with 4 independently failing
   components

baseball.pomdp.gz

   simple model of 1 inning of a baseball game

aloha.10.pomdp.gz

   slotted aloha model with a limit of 10 backlogged messages

aloha.30.pomdp.gz

   slotted aloha model with a limit of 30 backlogged messages
