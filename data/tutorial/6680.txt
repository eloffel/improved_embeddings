readme.md

enhanced lstm for natural language id136

   source code for "enhanced lstm for natural language id136" runnable
   on gpu and cpu based on theano. if you use this code as part of any
   published research, please acknowledge the following paper.

   "enhanced lstm for natural language id136" qian chen, xiaodan zhu,
   zhenhua ling, si wei, hui jiang, diana inkpen. acl (2017)
@inproceedings{chen-qian:2017:acl,
  author    = {chen, qian and zhu, xiaodan and ling, zhenhua and wei, si and jia
ng, hui and inkpen, diana},
  title     = {enhanced lstm for natural language id136},
  booktitle = {proceedings of the 55th annual meeting of the association for com
putational linguistics (acl 2017)},
  month     = {july},
  year      = {2017},
  address   = {vancouver},
  publisher = {acl}
}

   homepage of the qian chen, [75]http://home.ustc.edu.cn/~cq1231/

   the code is modified from [76]github - nyu-dl/dl4mt-tutorial.

   the code for tree-lstm version has been released. tree-lstm part is
   modified from [77]github - dallascard/treelstm, but support
   minibatches.

dependencies

   to run it perfectly, you will need:
     * python 2.7
     * theano 0.8.2

running the script

    1. download and preprocess

cd data
bash fetch_and_preprocess.sh

    2. train and test model for esim

cd scripts/esim/
bash train.sh

    3. train and test model for treelstm-im

cd scripts/treelstm-im/
bash train.sh

   the result is in log.txt file.
