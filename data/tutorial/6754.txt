   name description

   [110]f4de-3.5.0
   framework for detection evaluations (includes clear, trecvid event
   detection, and avss multi-camera person tracking evaluation tools)
   [111]madcateval_1.1.0-20100201-1609.tar.bz2 ([112]md5) madcat
   evaluation toolkit version 1.1.0
   [113]mteval-v13a-20091001.tar.gz mteval scoring script for the open mt
   evaluation, with example. version 13a.
    [114]https://github.com/usnistgov/sctk id103 scoring
   toolkit (sctk); includes the sclite, asclite, tranfilt, hubscr,
   slatreport and utf_filt scoring tools).
   [115]stdeval-0.6-20061122-1108.tgz spoken term detection (std)
   evaluation toolkit version 0.6 (includes the stdeval, detutil and
   procgraph).
   [116]score_3.6.2.tgz id103 scoring package(score) version
   3.6.2
   [117]detware_v2.1.tar.gz det-curve plotting software for use with
   matlab.
   [118]gnu_detware.tgz det-curve plotting software for use with gnuplot.
   [119]ieeval0.6.tar.gz 1998 information extraction - named entity
   (ie-ne) scoring software version 0.6 contributed by saic.
   [120]edt_.scorer.tar.gz developed for ace (edt phase 1). compares
   different alembic annotations for edt phase1 task definition.
   comparison is performed on alembic xml output files.
   [121]tranfilt-1.14.tgz a set of asr transcription filtering scripts.
   these scripts are used in conjunction with sctk to score the nist
   id103 evaluations.
   [122]seg_scr.v21.tgz generic time-based segmentation scorer with
   examples. developed for the 2000 nist speaker recognition evaluation.
   [123]tdt3eval_v2.6 evaluation script suite for the tdt evaluations

miscellaneous tools

   name description
   [132]jobrunner-1.3.0-20130404-2047z.tar.bz2 ([133]md5) jobrunner is a
   tool designed to run a job (ie to execute a command line or script)
   that does not require user interaction, and stores the program's output
   (standard error and standard out) in an easy to read log file. the tool
   is able to save a configuration file of the job to be run, that can be
   used by jobrunner_caller as a light-weight queueing system.
