gans-awesome-applications

   curated list of awesome gan applications and demonstrations.

   note: general gan papers targeting simple image generation such as
   dcgan, began etc. are not included in the list. i mainly care about
   applications.

the landmark papers that i respect.

     * id3, [76][paper], [77][github]
     * unsupervised representation learning with deep convolutional
       id3, [78][paper], [79][github]
     * improved techniques for training gans, [80][paper], [81][github]
     * began: boundary equilibrium id3,
       [82][paper], [83][github]

applications using gans

font generation

     * learning chinese character style with conditional gan, [113][blog],
       [114][github]

anime character generation

     * towards the automatic anime characters creation with generative
       adversarial networks, [115][paper]
     * [project] a simple pytorch implementation of generative adversarial
       networks, focusing on anime face drawing, [116][github]
     * [project] a simple, clean tensorflow implementation of generative
       adversarial networks with a focus on modeling illustrations,
       [117][github]
     * [project] keras-gan-animeface-character, [118][github]
     * [project] a dcgan to generate anime faces using custom mined
       dataset, [119][github]

interactive image generation

     * generative visual manipulation on the natural image manifold,
       [120][paper], [121][github]
     * neural photo editing with introspective adversarial networks,
       [122][paper], [123][github]

text2image (text to image)

     * tac-gan     text conditioned auxiliary classifier generative
       adversarial network, [124][paper], [125][github]
     * stackgan: text to photo-realistic image synthesis with stacked
       id3, [126][paper], [127][github]
     * generative adversarial text to image synthesis, [128][paper],
       [129][github], [130][github]
     * learning what and where to draw, [131][paper], [132][github]

3d object generation

     * parametric 3d exploration with stacked adversarial networks,
       [133][github], [134][youtube]
     * learning a probabilistic latent space of object shapes via 3d
       generative-adversarial modeling, [135][paper], [136][github],
       [137][youtube]
     * 3d shape induction from 2d views of multiple objects, [138][paper]
     * fully convolutional refined auto-encoding generative adversarial
       networks for 3d multi object scenes, [139][github], [140][blog]

image editing

     * invertible conditional gans for image editing, [141][paper],
       [142][github]
     * image de-raining using a conditional generative adversarial
       network, [143][paper], [144][github]

face aging

     * age progression/regression by conditional adversarial autoencoder,
       [145][paper], [146][github]
     * can: creative adversarial networks generating    art    by learning
       about styles and deviating from style norms, [147][paper]
     * face aging with conditional id3,
       [148][paper]

human pose estimation

     * pose guided person image generation, [149][paper]

domain-transfer (e.g. style-transfer, pix2pix, sketch2image)

     * image-to-image translation with conditional adversarial networks,
       [150][paper], [151][github], [152][youtube]
     * unpaired image-to-image translation using cycle-consistent
       adversarial networks, [153][paper], [154][github], [155][youtube]
     * learning to discover cross-domain relations with generative
       adversarial networks, [156][paper], [157][github]
     * unsupervised creation of parameterized avatars, [158][paper]
     * unsupervised cross-domain image generation, [159][paper]
     * precomputed real-time texture synthesis with markovian generative
       adversarial networks, [160][paper], [161][github]
     * pixel-level domain transfer [162][paper], [163][github]
     * texturegan: controlling deep image synthesis with texture patches,
       [164][paper], [165][demo]
     * vincent ai sketch demo draws in throngs at gtc europe, [[166]blog],
       [167][youtube]
     * deep photo style transfer, [168][paper], [169][github]

image inpainting (hole filling)

     * context encoders: id171 by inpainting, [170][paper],
       [171][github]
     * semantic image inpainting with perceptual and contextual losses,
       [172][paper], [173][github]
     * semi-supervised learning with context-conditional generative
       adversarial networks, [174][paper]
     * generative face completion, [175][paper], [176][github]

super-resolution

     * image super-resolution through deep learning, [177][github]
     * photo-realistic single image super-resolution using a generative
       adversarial network, [178][paper], [179][github]
     * high-quality face image super-resolution using conditional
       id3, [180][paper]
     * analyzing perception-distortion tradeoff using enhanced perceptual
       super-resolution network, [181][paper], [182][github]

image blending

     * gp-gan: towards realistic high-resolution image blending,
       [183][paper], [184][github]

high-resolution image generation (large-scale image)

     * generating large images from latent vectors, [185][blog],
       [186][github]
     * progressive growing of gans for improved quality, stability, and
       variation, [187][paper], [188][github]

adversarial examples (defense vs attack)

     * safetynet: detecting and rejecting adversarial examples robustly,
       [189][paper]
     * adversarial examples for generative models, [190][paper]
     * adversarial examples generation and defense based on generative
       adversarial network, [191][paper]

visual saliency prediction (attention prediction)

     * salgan: visual saliency prediction with generative adversarial
       networks, [192][paper], [193][github]

id164/recognition

     * perceptual id3 for small object
       detection, [194][paper]
     * adversarial generation of training examples for vehicle license
       plate recognition, [195][paper]

robotics

     * unsupervised pixel   level id20 with generative
       adversarial networks, [196][paper], [197][github]

video (generation/prediction)

     * deep multi-scale video prediction beyond mean square error,
       [198][paper], [199][github]

synthetic data generation

     * learning from simulated and unsupervised images through adversarial
       training, [200][paper], [201][github]

others

     * (physics) learning particle physics by example: location-aware
       id3 for physics synthesis,
       [202][paper], [203][github]
     * (games) style transfer id3: learning to
       play chess differently, [204][paper], [205][github]
     * (general) spectral id172 for generative adversarial
       networks, [206][paper], [207][github]

did not use gan, but still interesting applications.

real-time face reconstruction

     * model-based deep convolutional face autoencoder for unsupervised
       monocular reconstruction, [208][paper], [209][github],
       [210][youtube]

super-resolution

     * learning to simplify: fully convolutional networks for rough sketch
       cleanup, [211][paper], [212][site link], [213][youtube]

photorealistic image generation (e.g. pix2pix, sketch2image)

     * the sketchy database: learning to retrieve badly drawn bunnies,
       [214][paper], [215][youtube]
     * patchmatch: a randomized correspondence algorithm for structural
       image editing, [216][paper], [217][github], [218][youtube]

human pose estimation

     * knowledge-guided deep fractal neural networks for human pose
       estimation, [219][paper], [220][github]

3d object generation

     * 3d-r2n2: a unified approach for single and multi-view 3d object
       reconstruction, [221][paper], [222][github]
