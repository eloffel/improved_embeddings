readme.md

swish-activation

   this repo contains code for replicating the results from my two part
   blog post about swish activation function published on medium. folder
   mnist contains code and results for part 1 and cifar-10 for part 2.

   part 1: experiments with swish activation function on mnist dataset
   [72]https://medium.com/@jaiyamsharma/experiments-with-swish-activation-
   function-on-mnist-dataset-fc89a8c79ff7

   part 2: swish in depth: a comparison of swish & relu on cifar-10
   [73]https://medium.com/@jaiyamsharma/swish-in-depth-a-comparison-of-swi
   sh-relu-on-cifar-10-1c798e70ee08
